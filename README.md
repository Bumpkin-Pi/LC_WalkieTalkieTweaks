# LC_WalkieTalkieTweaks
Simple mod to reduce (or increase) the battery life of walkie talkies in Lethal Company.

Why? because having constant coms completely removes the horror element. This way you have to be sparing with your walkie talkie usage.

## Install instructions:
1. Have BeInEx installed.
2. Download LC_WalkieTalkieTweaks.dll (from releases).
3. Put it into `[STEAM FOLDER]/steamapps/common/Lethal Company/BepInEx/plugins`

### In the case of r2modman or other mod loaders:
1. Download LC_WalkieTalkieTweaks
2. Place inside the `[MOD LOADER GAME INSTANCE]/BepInEx/plugins` folder


## Compile:
1. Dont
2. Just dont
3. `git clone https://github.com/Bumpkin-Pi/LC_WalkieTalkieTweaks.git`
4. `cd LC_WalkieTalkieTweaks`
5. `dotnet build .`
